SpinWheelMaker.com

Spinwheelmaker.com is a full stack app (MERN Stack) that allows users to create their own custom spin wheel to make a choice at random for them. A user adjust the settings of the spin wheel and even rearrange the order of the inputs using drag n drop. Once everything is finished a user can create a profile to save their work for later.
